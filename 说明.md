# claude-hook-monitor

Claude Code Hook 事件监听器 — 被动记录所有 18 个 hook 事件，以结构化日志输出。

## 环境要求

- Rust toolchain（推荐通过 [rustup](https://rustup.rs/) 安装）
- cargo（随 Rust 一起安装）

## 开发环境运行

```bash
# 进入项目目录
cd 氛围桌搭-hook

# 直接运行（cargo run 会自动编译）
cargo run -- cat
cargo run -- gui #带窗口 和猫
cargo run -- listen
cargo run -- status
cargo run -- tail
cargo run -- tail --filter PreToolUse
cargo run -- install --scope user
cargo run -- install --scope project

# 模拟 hook 事件测试
printf '{"hook_event_name":"PreToolUse","session_id":"test","transcript_path":"/tmp/t","cwd":"/tmp","permission_mode":"default","tool_name":"Bash","tool_input":{"command":"npm test"}}' | cargo run -- listen

printf '{"hook_event_name":"SessionStart","session_id":"test","transcript_path":"/tmp/t","cwd":"/tmp","permission_mode":"default","source":"startup","model":"claude-sonnet-4-6"}' | cargo run -- listen

printf '{"hook_event_name":"PermissionRequest","session_id":"test","transcript_path":"/tmp/t","cwd":"/tmp","permission_mode":"default","tool_name":"Bash","tool_input":{"command":"rm -rf node_modules"}}' | cargo run -- listen

# 查看已记录的事件
cargo run -- status
```

## 编译

```bash
# Debug 编译（快，体积大，含调试信息）
cargo build

# Release 编译（慢，体积小，性能优化）
cargo build --release
```

编译产物位置：

| 模式 | 路径 |
|------|------|
| Debug | `target/debug/claude-hook-monitor` |
| Release | `target/release/claude-hook-monitor` |

## 安装使用

```bash
# 1. Release 编译
cargo build --release

# 2.（可选）复制到 PATH 中
cp target/release/claude-hook-monitor /usr/local/bin/

# 3. 安装 hook 配置
#    --scope user    写入 ~/.claude/settings.json（所有项目生效）
#    --scope project 写入 .claude/settings.json（仅当前项目）
claude-hook-monitor install --scope user

# 4. 重启 Claude Code 加载新 hooks

# 5. 另开终端，实时查看事件流
claude-hook-monitor tail
```

## 子命令说明

| 命令 | 说明 |
|------|------|
| `listen` | Hook 处理器，被 Claude Code 通过 stdin 调用，用户一般不直接使用 |
| `install --scope <user\|project>` | 生成 hook 配置写入 settings.json |
| `tail [--filter <事件类型>]` | 实时查看事件日志，支持按类型过滤 |
| `status` | 显示最近事件的统计摘要 |

## 日志位置

```
~/.claude-hook-monitor/events.jsonl
```

格式为 JSON Lines，每行一个事件，可用 jq 等工具分析：

```bash
# 查看最近 10 条
tail -10 ~/.claude-hook-monitor/events.jsonl | jq .

# 统计各事件类型数量
cat ~/.claude-hook-monitor/events.jsonl | jq -r .event_type | sort | uniq -c | sort -rn

# 只看 PreToolUse 事件
cat ~/.claude-hook-monitor/events.jsonl | jq 'select(.event_type == "PreToolUse")'
```

```
│   ├── claude-code-hooks.md         # 18 个 Hook 事件完整参考文档

```
